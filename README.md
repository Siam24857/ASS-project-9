# WEbsite name:  StudyNook 

Live Site URL: [https://ass-project-9.vercel.app] 

## Features

-  Browse & Search** - Find study rooms by name, amenities, and price range
-  Instant Booking** - Book study rooms with real-time availability checking
- User Dashboard** - Manage your bookings and room listings from one place
-  Host Your Room** - List your own study space and earn hourly rates
-  Fully Responsive** - Seamless experience on mobile, tablet, and desktop

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Better Auth
- MongoDB

## Quick Start

bash
npm install
npm run dev
